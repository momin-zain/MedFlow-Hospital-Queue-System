# MedFlow - Hospital Queue Management System

## 🏥 Project Overview
MedFlow is a real-time hospital queue orchestration and analytics ecosystem that digitizes the entire patient flow process from registration to consultation completion.

## ✨ Features
- **Reception Dashboard** - Add patients, generate tokens, manage queue
- **Public Display** - Live queue status for waiting area
- **Doctor Portal** - Secure login, manage assigned patients
- **Analytics Dashboard** - Charts for patient volume, doctor workload, peak hours
- **Appointment Booking** - Online pre-booking with time slots
- **Settings Module** - Configure hospital info, doctors, working hours

## 🛠️ Technology Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6)
- **Database**: Firebase Firestore (Real-time)
- **Charts**: Chart.js
- **Icons**: Font Awesome 6
